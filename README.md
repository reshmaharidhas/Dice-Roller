Basic dice roller application which can roll 6 different types of dices in your desktop. This application has
 - Tetrahedron dice
 - Cubic dice
 - Octahedron dice
 - Pentagonal trapezohedron dice
 - Dodecahedron dice
 - Icosahedron dice
